# Love Tracker

## Features
- Track romantic interests
- Monitor relationship milestones
- Discover meaningful quotes

## Deployment Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Riverbrookey/love-tracker.git
   cd love-tracker
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Setup GitHub Pages
1. Go to the repository settings.
2. Under "Pages", select the branch you want to use (usually `main`) and set the root folder.
3. Save the settings and your site will be published at `https://<username>.github.io/love-tracker/`.

## Note
Make sure you have the necessary environment variables set up for the application to run successfully.
